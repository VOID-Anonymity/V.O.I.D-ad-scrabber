## 🌑 Суть проекта (The Purpose)

В эпоху тотальной слежки, где каждый твой шаг фиксируется "пикселями" и метриками, **V.O.I.D ad scrabber** выступает в роли невидимого стража. 

Скрипт прочесывает самые зараженные уголки сети — от обманчивых развлекательных порталов до зловонных казино, — вырывая из их кода домены, которые служат для доставки рекламы и кражи данных. 

**Артефакт обновляется каждые 48 часов**, впитывая в себя свежие тени новых рекламных угроз.

## ⚔️ Использование (Ritual of Integration)

Ты можешь призвать силу **V.O.I.D ad scrabber** в любой свой инструмент защиты. Список готов к интеграции через прямую ссылку (Raw).

### 🗡️ Для uBlock Origin / AdGuard:
```
1. Открой настройки вашего блокировщика.
2. Перейди во вкладку **"Списки фильтров"** (Filter lists).
3. Найди раздел **"Импорт"** или **"Пользовательские"**.
4. Вставь следующую ссылку:
   `https://raw.githubusercontent.com/твой_профиль/v.o.i.d-ad-scrabber/main/dist/void_hosts.txt`
5. Нажми "Применить изменения".
```

### 🏰 Для OpenWrt (AdBlock / Simple AdBlock):
```
1. Зайди в интерфейс роутера (LuCI).
2. Перейди в настройки **Services -> AdBlock**.
3. Добавь новый источник в **"Blocklist Sources"**.
4. Укажи ссылку на Raw файл и сохрани. Теперь твой замок защищен на уровне врат.
```

### 🌌 Для DNS-серверов (Pi-hole / NextDNS / AdGuard Home):

Просто добавь ссылку в список Ad-листов. Бездна закроет доступ к рекламным мирам для всех устройств в твоей сети.

## 🔥 Как работает Ткач Бездны (Process)

1. **Extraction (Извлечение):** Используя мощные алгоритмы, мы перехватываем каждый скрытый сетевой запрос на самых агрессивных площадках интернета.
2. **Purification (Очистка):** Домены проходят через фильтры Бездны. Дубликаты и ложные цели стираются, остаются только истинные паразиты.
3. **Synthesis (Синтез):** Раз в двое суток создается кристально чистый список в формате `hosts`, готовый к немедленному использованию.
